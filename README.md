# Foro DarkZone


La aplicación trata de un foro temático de terror, donde los usuarios pueden subir sus blogs y revisar blogs creados por otros usuarios. Aparte también incluye una sección de noticias relevantes sobre temas terrorificos.

Este proyecto fue realizado por Catalina Rojas y Alejandra Faguas.

Funcionalidades implementadas:

1.Autenticación de Usuarios:
	Inicio de Sesión: Los usuarios pueden iniciar sesión con su nombre de usuario y contraseña.
	Registro: Los nuevos usuarios pueden registrarse proporcionando un nombre de usuario, correo electrónico y contraseña.

2.Gestión de Blogs:

	Creación y Edición: Los usuarios pueden crear nuevas publicaciones de blog, incluyendo un título, descripción y una imagen.
	Listado de Blogs: La pantalla principal de "Blogs" muestra todas las publicaciones en formato de lista.
	Detalle de Blog: Al hacer clic en una publicación, el usuario puede ver su contenido detallado.

3.Gestión de Perfil de Usuario:

	Visualización y Edición: Los usuarios pueden ver y actualizar su nombre de usuario, correo electrónico y contraseña.
	Eliminación de Cuenta: Se ofrece la opción de eliminar la cuenta de forma permanente.

4.Navegación:
	
	Menú Lateral (Navigation Drawer): Un menú de navegación lateral permite al usuario moverse entre las diferentes secciones de la aplicación: Inicio, Blogs, Perfil y Noticias.
	Cierre de Sesión: Desde el menú, el usuario puede cerrar su sesión, lo que lo redirige a la pantalla de inicio de sesión.

5.Otras Pantallas: 

	Noticias: Una pantalla para mostrar noticias.

Pasos para Ejecutar y Flujo de la Aplicación

1.Inicio y Autenticación:
	
	La aplicación se inicia en la pantalla de Login.
	Si el usuario no tiene una cuenta, puede navegar a la pantalla de Registro para crear una.
	Una vez registrado, vuelve a la pantalla de Login.

2.Acceso a la Aplicación:
	
	Tras un inicio de sesión exitoso, el usuario es dirigido a la pantalla principal de la aplicación, que por defecto es la pantalla de Inicio.

3.Navegación y Uso:

	El usuario puede abrir el menú de navegación lateral para acceder a:
	Blogs: Para ver, crear y leer publicaciones.
	Perfil: Para gestionar los datos de su cuenta.
	Noticias: Para ver las últimas noticias.
	Desde el menú, también puede Cerrar Sesión en cualquier momento, lo que lo devolverá a la pantalla de Login.
